# 02 · 信息架构与导航

## 1. 层级结构

```
App
└── 作品列表（Home）
    └── 作品工作台（Project Workspace）
        ├── [全局] 角色名侧栏（Layer 0，最左）
        ├── [全局] 主导航侧栏（Layer 1）
        └── [内容区] 当前模块（Layer 2）
            ├── 正文
            ├── 地图
            ├── 角色
            └── 设定
```

## 2. 路由设计（建议）

| 路径 | 页面 | 说明 |
|------|------|------|
| `/` | 作品列表 | 新建/打开/最近作品 |
| `/project/:id` | 重定向 | → `/project/:id/manuscript` |
| `/project/:id/manuscript` | 正文 | 默认落地页 |
| `/project/:id/manuscript/:chapterId` | 正文·指定章 | 深链接 |
| `/project/:id/map` | 地图 | |
| `/project/:id/characters` | 角色 | 关系网视图 |
| `/project/:id/characters/:characterId` | 角色详情 | 可抽屉或全页 |
| `/project/:id/settings` | 设定 | 自定义模块 |
| `/project/:id/settings/:moduleId` | 设定·单模块锚点 | 可选 |

## 4. 导航行为规则

### 4.1 角色名列表（全局）
- **所有模块下可见**（地图/角色/设定/正文）
- 点击角色名：
  - 若在「角色」页 → 居中并高亮关系网节点
  - 若在「正文」页 → 右侧预览若含该角色则滚动到对应 diff 项
  - 其他页 → 打开角色详情抽屉（不强制跳转）
- 搜索：实时过滤，支持拼音首字母（中文名，可选）

### 4.2 主导航
- 图标 + 文字垂直排列
- 当前项高亮
- 切换导航**不丢失**正文未保存草稿（local draft）

### 4.3 首页 vs 工作台
- **首页**：作品列表 + 新建向导（书名、主角名、地图提示词）
- **进入作品后**：默认落地「正文」；地图在「地图」导航查看
- **存储目录**：顶栏全局可见，用户自选，切换后重新加载作品列表

**已确认（A3）：** 不采用「进入作品先看地图大屏」方案。

## 5. 状态与上下文

### 跨页共享状态
| 状态 | 作用域 | 持久化 |
|------|--------|--------|
| 当前作品 ID | 全局 | 是 |
| 选中角色 ID | 全局 | URL 可选 |
| 正文草稿 | 章节级 | 自动保存 local |
| 预览暂存（未 commit） | 章节级 | **仅内存**，不入库 |
| 侧栏折叠状态 | 用户偏好 | localStorage |

### 深链接场景
- 分享角色卡：`/project/:id/characters/:characterId`
- 从关系网复制链接打开直接定位节点

## 6. 空状态设计

| 场景 | 引导 |
|------|------|
| 无作品 | 大按钮「创建第一部作品」+ 简短说明 |
| 无章节 | 正文区「粘贴第一章」或「上传文件」 |
| 无角色 | 识别后自动生成；或手动「添加角色」 |
| 无地图 | 提示填写地图说明并「AI 生成」 |
| 无设定模块 | 设定页「添加第一个模块」+ 工具选择 |

## 7. 快捷键（建议）

| 快捷键 | 动作 |
|--------|------|
| `Ctrl+S` | 保存当前章节正文 |
| `Ctrl+Shift+R` | 智能识别 |
| `Ctrl+K` | 聚焦角色搜索 |
| `Ctrl+\` | 折叠/展开角色名侧栏 |
| `Ctrl+Enter` | 预览区「确认更新」（焦点在预览时） |
