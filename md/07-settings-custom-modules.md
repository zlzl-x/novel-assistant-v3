# 07 · 设定页与自定义模块

## 1. 设计理念

> **我们只提供工具，不提供「正确的小说设定格式」。**

作者创作习惯差异极大：
- 有人先写百万字再补大纲
- 有人先建完整力量体系再动笔
- 有人用思维导图，有人只用自由笔记

设定页 = **可组合的块编辑器（Block Editor）**。

## 3. 模块类型

### 3.1 富文本块（MVP）
- 自定义标题
- 内容：Markdown 或轻量 WYSIWYG
- 支持标题、列表、粗体、链接
- 可 `@` 引用角色名、地图节点（插入只读链接）

### 3.2 清单块（MVP）
- 待办/设定核对清单
- 例：「伏笔回收列表」「待登场角色」

### 3.3 表格块（MVP+1）
- 自定义列（文本、数字、单选）
- 预设模板：境界表、功法表、势力表

### 3.4 思维导图块（第二期）
**轻量方案：**
- 嵌入开源组件（如 `mind-elixir`、`simple-mind-map`）
- 节点纯文本，导出 JSON 存模块内

**重量方案：**
- 外链到 XMind / Obsidian（不做）

**功能范围：**
- 添加/删除节点
- 拖拽布局
- 导出 PNG（可选）
- 不做：协同、演示模式

### 3.5 关联摘要块（二期）
- 选择角色或地图节点
- 显示其 current 字段只读快照
- 设定页改动不影响源数据

### 3.6 时间线块（二期）
- 按卷/章排列大事记
- 可链接章节

## 4. 工具栏 · 添加模块流程

```mermaid
flowchart LR
    A[点击 + 添加模块] --> B[选择类型]
    B --> C[输入标题]
    C --> D[插入页面底部]
    D --> E[焦点落入内容区]
```

**模板库（MVP 四模板全部内置，内容可空白）：**
| 模板名 | 包含 |
|--------|------|
| 大纲（起承转合） | 富文本 + 预设标题结构 |
| 力量体系 | 表格块 + 空白行 |
| 人设表 | 表格块 + 空白行 |
| 伏笔追踪表 | 清单块 + 空白项 |

模板 = 一次插入对应块，**内容可为空**，可删除可改。

## 5. 模块操作

| 操作 | 说明 |
|------|------|
| 拖拽排序 | 左侧把手 ≡ |
| 折叠 | 只显示标题栏 |
| 重命名 | 点击标题编辑 |
| 删除 | 二次确认 |
| 复制模块 |  DUPLICATE 到本书或另一本书 |
| 导出单模块 | Markdown |

---

## 6. 与识别系统的联动（可选增强）

- 设定模块「力量体系」表格 → 注入 LLM 作为境界合法值参考
- 设定模块「势力列表」→ 助势力字段归一
- **不自动**从设定同步到角色库（避免双向混乱）

## 8. 思维导图工具补充说明

### 8.1 适用场景
- 剧情分支
- 势力关系（与角色关系网互补：设定层 vs 数据层）
- 伏笔线索

### 8.2 与「角色关系网」边界

| | 角色关系网 | 思维导图模块 |
|--|-----------|--------------|
| 数据 | 结构化 Character | 自由节点 |
| 来源 | 识别 + 手动 | 纯手动 |
| 位置 | 角色导航 | 设定导航 |
| 用途 | 防吃书查阅 | 创作思考 |

### 8.3 实现建议
- **MVP 不做思维导图**（已确认 D1），用「富文本 + 缩进列表」临时替代
- 二期评估 `mind-elixir` 集成成本
